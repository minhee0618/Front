
자바스크립트 만나기
  

   1..  자바스크립트로 무엇을 하는지 알아보기 

         1. 사용자의 입력및 계산  
         2. 유효성체크 
         3. 이벤트 처리 
         4. 동적으로  css 변경  
         5. 서버와의 통신   
   
       - 롯데(마이롯데) 회원, 비회원에 따라 화면이 달라짐 
         한 화면에 두 개의 영역이 있음 
         회원일때만 보이는 영역 , 비회원일 때만 보이는 영역  :  display:none, display:block
       
       - 장바구니 계산

   2..  자바스크립트 언어 소개  ( 목적: 웹의 실시간 유효성체크 )

   3.. 자바스크립트 실행
         
       1. html내에서 실행  <script> </script>내 - <head>안, <body>안
       2. 개발자도구에서 실행 (f12) -- Ctrl+L 콘솔 지우기
       3. node 설치후 실행

   4..  자바스크립트 코드 작성 위치   
   
      1. 웹문서 내부 <script></script>
      2. 외부 자바스크립트 파일로 작성 <script src=""></script>
      3. <a href="javascript:alert('click');">클릭</a>
      4. <button onclick="alert('click');">클릭</button>
  
   5.. 자바스크립트 입출력 도구
      1. prompt : 입력창을 뛰워 사용자 입력 받기
      2. alert : 알림창 띄우기
      3. confirm : 확인/취소 선택창 띄우기
      4. document.write  : 웹페이지에 html 출력 
      

  -  자바스크립트 문법
  -  자료형  , 변수 , 리터럴  , 연산자,  제어문  , 함수, 배열 , 객체
  -  (숫자형, 문자형, 논리형, 객체, undefined , null)
  -  자료형과 변수이야기
