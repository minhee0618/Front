시멘틱 태그 : 의미적 태그

div 태그와 동일. 기능x (의미만 부여)

시멘틱 태그는 html요소의 의미를 명확하게 나타내는 태그

시멘틱 태그를 사용하면 문서 구조를 이해하기 쉽고
검색엔진이 페이지의 컨텐츠를 웹 접근성을 향상 시키고 스타일/레이아웃 작업을 쉽게 수행 가능

  • header : 헤더
  • nav : 네비게이션 링크 영역
  • main : 주요 컨텐츠 포함 영역
  • section : 문서의 일반적 섹션, 그룹핑
  • article : 독립적 컨텐츠 조작
  • aside : 주 내용과 연관성이 적은 컨텐츠
  • footer : 문서의 바닥글, 저작권 정보, 연락처
