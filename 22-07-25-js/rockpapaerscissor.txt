가위바위보 체계적인 문제풀이


사용자와 컴퓨터가 가위바위보하는 프로그램을 작성하려고 한다. 
가위,바위,보는 1,2,3의 숫자로 나타냅니다.
컴퓨터는 랜덤으로 손을 선택합니다. 결과화면에는 ‘무승부’, ‘사용자승리’ ,‘사용자패배’ 중 하나를 표시합니다. 무승부의 경우는 다시 승부합니다.
 (승패가 나면 사용자의  승리 또는 패배를 출력한다 )

 가위 1 할당
 바위 2 할당
 보 3 할당
 컴퓨터 1,2,3중 랜덤 선택
 사용자 1,2,3중 선택
 C 1 U 1 = 무승부 / C 2 U 2 = 무승부 / C 3 U 3 = 무승부
 C 1  U 2 = / C 1 U 3
 C 2  U 1 / C2/ C2 U3
 A-1 

0.문제 분석(입력, 출력, 처리)
 1.자료의 명세(데이터, 변수찾기)
 2.처리과정(대략적인 순서, what분석- 세분화)
 3.순서도 그리기
 4.검증
 
 프로그램이란?
 데이터+ 기능(함수) => 명령

 ######
 ##자료의 명세 (데이터 찾기, 변수 칮기)

 입력(데이터) 변수; 사용자가 입력한 가위 바위 보: input
 처리(데이터) 변수; 컴퓨터가 임의로 낸 수: computer
 출력(데이터) 변수; 사용자 승패: result

 ##처리과정  (반복을 중심으로 처리과정 작성)
  - 반복에서 해야 할 일/ 반복이 끝나고 해야 할 일 구분 중요


 1.반복 (승패가 날때까지, 무승부면 반복)
  1.1 사용자가 가위바위보를 낸다
  1.2 컴퓨터가 가위바위보를 임의로 정한다
  1.3 사용자와 컴퓨터가 내 수를 판단한다

 2.사용자의 승패를 판단한다
  2.1 사용자가 가위 일때
    - 컴퓨터가 보면 사용자 승
  2.2 사용자가 바위 일때
    - 컴퓨터가 가위이면 사용자 승
  2.3 사용자가 보 일때
    - 컴퓨터가 주먹이면 사용자 승

3.사용자 승패 출력
