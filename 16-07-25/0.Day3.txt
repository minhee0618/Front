6. 마음에 드는 웹사이트의 푸터(Footer) 영역을 참고하여 직접 구현해보는 경험을 할 수 있다.

HTML의 모든 태그는 기본적으로 block(블록 요소), inline(인라인 요소), inline-block 중 하나의 디스플레이 속성을 가지고 있다
이러한 속성은 태그의 "태생적인" 디폴트(display) 값으로 정해져 있지만, 필요에 따라 CSS의 display 속성을 사용해 변경할 수 있다.


######
 HTML 요소의 디스플레이 방식(CSS의 display 속성 값)
css  display 속성:      block,  inline ,  inline-block   
                 block                            inline                      inline-block
 줄 바꿈 여부:  	 줄 바꿈 (새 줄에서 시작)        	줄 바꿈 없음	                줄 바꿈 없음
 크기 설정  :	    가능 (width, height)	         불가능	                       가능
 여백 적용	:      모든 방향 적용 가능   	         상하 여백은 무시됨	             모든  방향 적용 가능

#####박스모델####
모든 HTML 태그는 브라우저에서 '박스(Box)' 형태로 표현된다.
이 박스는 콘텐츠 영역을 중심으로 패딩(Padding), 보더(Border),  마진(Margin) 으로 구성되며, 이를 박스 모델(Box Model)이라고 한다.


**Box의 구성요소

Padding: 콘텐츠와 테두리 사이의 안쪽 여백
Border: 패딩을 감싸는 테두리
Margin: 박스 외부와의 간격을 조절하는 바깥 여백
Content: 실제 내용이 들어가는 영역
